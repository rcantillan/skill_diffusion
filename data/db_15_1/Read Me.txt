O*NET 15.1 Database
February 2011 Release
---------------------

The current release of the O*NET database is available online at:

http://www.onetcenter.org/database.html

For information on this special Taxonomy Preview release of the O*NET
database, visit:

http://www.onetcenter.org/db_preview.html


New in 15.1
-----------

The O*NET 15.1 database is the first to feature the O*NET-SOC 2010
taxonomy, which is compatible with the recently released 2010 SOC
standard. This release does not feature new data from the O*NET Data
Collection Program; the next set of updated occupations will be
delivered in O*NET 16.0, expected in July 2011.


For More Information
--------------------

For comprehensive information on the contents, structure, and
relationship of the database files, and a history of changes in each
database release, please consult the O*NET Data Dictionary 15.1:

http://www.onetcenter.org/dl_files/DataDictionary15_1.pdf
